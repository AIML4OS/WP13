# WP13 AT Example Code

This folder contains the Austrian WP13 example workflow for synthetic data
generation and evaluation.

- `Example_Code_WP13.qmd` is the documented Quarto version with headings,
  explanations, and labeled code chunks. It is intended for reading, rendering,
  and presenting the workflow.
- `Example_Code_WP13.R` is the pure R script version of the same workflow,
  intended for direct execution or reuse in R.
